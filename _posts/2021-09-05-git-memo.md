## 常用命令

查看所有分支及当前分支

```bash
git branch -a
```

提交修改记录

```bash
git add .
git commit -m "fix typo"
git push origin main  # main意指将修改提交到远程main分支
```

切换分支

```bash
git checkout dev  # 切换到dev分支
```

删除分支

```bash
git branch -d dev  # 删除本地dev分支
git push origin --delete dev  # 删除远程dev分支
```

撤销本地commit

```bash
git reset --soft HEAD^
```

撤销本地add

```bash
git reset HEAD .
```
